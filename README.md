# Git

کلون گرفتن 
```commandline
git clone ./link/
```

### اتصال به اکانت گیت هاب
وارد کردن نام کاربری
```commandline
git config --global user.name "Your Name"
```
وارد کردن ایمیل
```commandline
git config --global user.email "your.email@example.com"
```
 دیدن وضعیت فایل ها
```bash
git status
```

اد کردن همه  فایل ها 
```bash
git add .
```

اد کردن یک فایل خاص 
```bash
git add .\fileKhas
```

کامیت کردن فایل
```bash
git commit -m "commit_message"
```

پوش کردن فایل 
```commandline
git push origin main
```

پول کردن  
```commandline
git pull origin main
```

دیدن راهنمای استفاده از کامند های گیت
```commandline
git help
```

# Linux
دیدن فایل های موجود در پوشه
```commandline
ls
```

خارج شدن از پوشه
```commandline
cd ..
```

ورود به پوشه خاص
```commandline
cd .\folder's name\
```
